## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites
- A working Laptop or Desktop
- Internet
- Knowledge of React, Redux

### Setup
- Install code editor
- Install Git bash
- Install react app
- Install react-redux
### Install

- React:
npx create-react-app the-app-name
- Redux: 
npm install redux react-redux 

### Dependencies

### Clone Project
git@github.com:mutinhiri/hello-rails-frontend.git


### Run tests
- npm run start
- npm run test
